# Cybersecurity Awareness Dashboard

An educational desktop application built with **C#**, **Avalonia UI**, and **.NET 10** to train users in core cybersecurity hygiene. The application features a persistent task manager, an interactive quiz game, a stateful chatbot, and an audit log console.

## 🚀 Key Features (POE Rubric Compliance)

### 1. Task Assistant & SQLite Integration (15 Marks)
*   **Database Backend:** Uses a local **SQLite database** (`tasks.db`) via the `Microsoft.Data.Sqlite` driver, replacing flat-file JSON storage.
*   **Automatic Setup:** Schema validation automatically creates the `Tasks` table upon launching the app for the first time.
*   **Complete CRUD:** Fully supports creating, loading, marking as completed, and deleting tasks dynamically with secure parameterized SQL commands.

| Column | Data Type | Constraint | Description |
| :--- | :--- | :--- | :--- |
| **`Id`** | `TEXT` | `PRIMARY KEY` | Unique GUID generated by the application |
| **`Title`** | `TEXT` | `NOT NULL` | The name of the security task |
| **`Description`** | `TEXT` | None | Context details for the task |
| **`Reminder`** | `TEXT` | None | Target deadline or date |
| **`Status`** | `TEXT` | `NOT NULL` | Tracks status (`'Pending'` or `'Completed'`) |

### 2. Interactive Cyber Quiz (15 Marks)
*   **Assessment:** Consists of **11 questions** testing core cyber topics like WPA3, MFA, phishing, public Wi-Fi, and firmware updates.
*   **GUI Feedback:** Answers trigger immediate color-coded visual feedback (Green for Correct, Red for Incorrect) with detailed educational explanations.
*   **Gamification Rank Output:** Evaluates final scores to print a customized cybersecurity rank in the UI:
    *   `🛡️ RANK: CYBER SENTINEL` (Score $\ge$ 9) — Strong defensive profile.
    *   `⚠️ RANK: STANDARD USER` (Score 5-8) — Basic protection active with minor vulnerabilities.
    *   `🛑 RANK: SECURITY RISK` (Score < 5) — Critical gaps requiring urgent review.
*   **Restart State:** Clicking restart flushes all choices and logs the metrics reset cleanly.

### 3. Stateful NLP ChatBot Core (10 Marks)
*   **Conversational Memory:** Supports multi-turn context tracking. Asking the bot to *"remind me to..."* triggers a state machine that prompts for a deadline and subsequently creates the task directly in the SQLite database.
*   **Response Variation:** Uses a randomized selection pool (`_randomEngine`) to deliver different tips regarding passwords or phishing so that the bot's outputs stay engaging.
*   **Sentiment & Intent Parsing:** Analyzes text inputs to route conversational flow or output system activity logs directly to the chat window.

### 4. Real-Time Activity Log (10 Marks)
*   **Audit Sidebar:** A persistent column on the right side of the dashboard logs background database, game, and chatbot events.
*   **Navigation:** Uses LINQ operators to limit the view to the **last 8 logs** to prevent clutter, with a **"Show Full Log"** button to expand the view to all session logs.

## 🛠️ Local Setup & Running Instructions

### Prerequisites
*   [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
*   VS Code with the C# extension, or Visual Studio 2022.

### How to Run:
1.  **Navigate to the project folder:**
    ```bash
    cd "Prog part1"
    ```
2.  **Restore the dependencies:**
    ```bash
    dotnet restore
    ```
3.  **Run the application:**
    ```bash
    dotnet run
    ```
    *Note: The SQLite database file (`tasks.db`) will generate automatically inside your running folder.*
